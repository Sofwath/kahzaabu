# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code
of conduct. The full text is canonical at:

> <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

The Contributor Covenant is licensed under
[CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) and is reproduced
in projects worldwide as the de-facto standard.

## The short version

Treat everyone with respect. Harassment, demeaning language, personal
attacks, doxxing, sustained disruption, sexualised attention, or
intimidation are not welcome — in issues, pull requests, commits, comments,
discussions, or any channel adjacent to this project (Telegram / WhatsApp /
Slack / Discord rooms wired via the hermes gateway included).

Subject-matter disagreement is fine; disagreement about how a person
identifies, is named, or is owed dignity is not.

## Scope

This applies to all project spaces — repository, issue tracker, PR
discussions, project events, and to any individual representing the
project in public.

## Enforcement

Report violations to: **Sofwathullah.Mohamed@gmail.com**

All reports are reviewed promptly and treated as confidential. The
project maintainer is responsible for clarifying standards, taking
appropriate corrective action, and may temporarily or permanently ban
contributors whose behaviour they deem inappropriate, threatening,
offensive, or harmful.

We follow the four-tier enforcement ladder from Contributor Covenant 2.1:

1. **Correction** — private, written warning.
2. **Warning** — formal warning with consequences.
3. **Temporary ban** — time-bounded exclusion from project spaces.
4. **Permanent ban** — exclusion from all project spaces.

See the linked Contributor Covenant text above for the full criteria
applied at each tier.
